First Time Setup — Run this SQL in Supabase

CREATE TABLE IF NOT EXISTS admin_config (
  key TEXT PRIMARY KEY,
  value JSONB,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

ALTER TABLE admin_config ENABLE ROW LEVEL SECURITY;

CREATE POLICY "optajob_admin"
  ON admin_config FOR ALL USING (true) WITH CHECK (true);
Steps: 1. Go to supabase.com → your Optajob project   2. Left sidebar → SQL Editor → New Query   3. Paste SQL above → click Run   4. Should say "Success. No rows returned"   5. Come back here → enter URL and key → Connect → Push Now
